### Execution and Running Details (Frontend)

To run the frontend of the project:

1. Clone the repository:

   ```
   git clone https://github.com/RGTrigger/hanuman-chatbot.git
   ```

2. Navigate into the project directory:

   ```
   cd hanuman-chatbot
   ```

3. Install the required dependencies:

   ```
   npm install
   ```

4. Start the development server:

   ```
   npm run dev
   ```

5. Open the application in a browser at:

   ```
   http://localhost:5173
   ```

---

### Backend Integration

The frontend communicates with a **Flask-based backend API** that is responsible for generating chatbot responses.
The backend handles all AI-related logic using **Retrieval-Augmented Generation (RAG)** and runs as a separate service.

This separation ensures:

* Better scalability
* Clean architecture
* Easier debugging and maintenance
